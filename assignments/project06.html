Project06

Due: Tue, Dec 5, 2023 at 11:59 PM to Github Classroom Assignment

Requirements

  1. Visual Appeal using CSS
    1. Using raw CSS or one of the popular frameworks such as Bulma or Pure make your search results more visually appealing. (Bootstrap is also ok but more involved)
    2. Use Go’s HTML templates to substitute your search results into an HTML snippet including style information.
  2. Implement one of these Information Retrieval features including two test cases
    1. Auto-complete on page <title>s using a Search Trie. Your search results must update as each keystroke is typed.
    2. Search for images using the contents of the alt attribute and display the image in search results
    3. Include a “snippet” of text surrounding a search term so users of your search engine can see the context around the term. You can see examples in public search engines.

Given

In lecture we will discuss:

  1. the syntax for HTML templates and their application in Go
  2. the structure of CSS selectors, properties, and values
  3. how to reference style sheets in your HTTP ResponseWriter

An example of the alt attribute: <img src="dog.jpeg" alt="my dog Goldy playing with her toy">

An example of snippets searching duckduckgo.com for “computer science”

Rubric

Since this project wraps the semester, code review meetings will not include any TODO recommendations

  1. Visual appeal
    1. Minimal effort (30 pts)
    2. Good effort (40 pts)
    3. Great effort (50 pts)
  2. Implement one of the three IR features given above (50 pts)
  3. Extra credit: implement two of the IR features
    1. Good effort (5 pts)
    2. Great effort (5 additional points)
