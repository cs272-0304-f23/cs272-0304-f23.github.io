    2. Since some sites have more documents than others, you may stop crawling after 10,000 documents

Given

  1. We will discuss the XML format and Go’s support for XML in lecture
  2. You’ll want to use the Go tags to assist with unmarshalling the XML
  3. Here’s an example Go data structure which can be used to unmarshal the XML given at http://usfca.edu/sitemap.xml (use View Source to the the raw XML) 
     type(
 	Sitemap struct {
         Loc     string `xml:"loc"`
         LastMod string `xml:"lastmod"`
     }

     SitemapIndex struct {
         Sitemaps []Sitemap `xml:"sitemap"`
     }
 )
  4. You can extend this to the <urlset> and <url> structures as shown at https://www.usfca.edu/sitemap.xml?page=1 (use View Source to see the raw XML)
